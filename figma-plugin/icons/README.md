# FigRoExporter Icons

Put 20x20px PNG icons here for the type buttons in the Layer Properties panel.

## Required Icons

| Filename | Icon | Description |
|----------|------|-------------|
| `frame.png` | Frame | Basic container Frame |
| `textbutton.png` | TextButton | Clickable text button |
| `textlabel.png` | TextLabel | Static text label |
| `imagebutton.png` | ImageButton | Clickable image button |
| `imagelabel.png` | ImageLabel | Static image display |
| `scroll.png` | Scroll | ScrollingFrame container |
| `vpf.png` | VPF | ViewportFrame |
| `ignore.png` | Ignore | Skip/don't export |

## Format

- Size: **20x20px** PNG with transparency
- Style: monochrome white icons (will be inverted when active)
- Alpha channel for transparency

## Example

```
icons/
├── frame.png
├── textbutton.png
├── textlabel.png
├── imagebutton.png
├── imagelabel.png
├── scroll.png
├── vpf.png
└── ignore.png
```
