How to Get API Key

  1. Go to Roblox Developer Hub
  2. Navigate to Open Cloud → API Keys
  3. Click Create API Key
  4. Give it a name (e.g., "FigRoExporter")
  5. Under Assets, grant read/write permissions
  6. Copy the generated key

Your key is stored locally in Figma and never shared.
